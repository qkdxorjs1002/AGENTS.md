# REPLTA Workflow Viewer

Read `.agent-workflow/` Markdown files quickly from the VSCode Activity Bar.

## Features

- Active Workflow view for `request.md`, `plan.md`, `task.md`, `speedwagon.md`, and `index.md`.
- Workflow Overview preview that combines active workflow documents in VSCode's built-in Markdown preview.
- Archives view for `.agent-workflow/archive/*/` folders.
- Automatic refresh when `.agent-workflow/**/*.md` changes.

## Usage

Open a workspace that contains `.agent-workflow/`, then use the Workflow icon in the Activity Bar.
